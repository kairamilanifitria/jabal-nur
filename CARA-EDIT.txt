CARA MEMBUKA DAN MENGEDIT FILE:
================================================================================

1. Klik kanan file "index.html"
2. Pilih "Open With" atau "Buka dengan"
3. Pilih Notepad++ atau text editor Anda
4. Edit teks yang Anda inginkan (lihat panduan di bawah)
5. Tekan Ctrl + S untuk Save
6. Buka file index.html di browser untuk melihat hasilnya
7. Refresh browser (F5) untuk melihat perubahan

üìç 8. MENGGANTI GOOGLE MAPS:
================================================================================

1. Buka Google Maps di browser
2. Cari lokasi yayasan Anda
3. Klik tombol "Share" atau "Bagikan"
4. Pilih tab "Embed a map" atau "Sematkan peta"
5. Copy kode yang diberikan
6. Cari bagian di index.html dengan komentar:
   <!-- [GANTI INI] Cara mendapatkan Google Maps embed: -->
7. Ganti SELURUH tag <iframe>...</iframe> dengan kode yang Anda copy

Contoh yang harus diganti:
DARI:
    <iframe src="https://www.google.com/maps/embed?pb=!1m18!..."></iframe>
    
KE:
    [Paste kode iframe dari Google Maps Anda]

üí° 12. TIPS & TRIK:
================================================================================

‚úÖ SELALU BACKUP FILE:
   Sebelum edit, copy file index.html dan rename jadi "index-backup.html"

‚úÖ EDIT SEDIKIT-SEDIKIT:
   Edit satu bagian ‚Üí Save ‚Üí Refresh browser ‚Üí Cek hasil

   Gunakan Ctrl + F untuk mencari teks yang ingin diedit

‚ùì TROUBLESHOOTING - MASALAH UMUM:
================================================================================

1. "Gambar tidak muncul"
   ‚ûú Pastikan nama file gambar SAMA PERSIS (case sensitive)
   ‚ûú Pastikan gambar ada di folder images/

2. "Layout berantakan setelah edit"
   ‚ûú Kemungkinan ada tag HTML yang terhapus
   ‚ûú Restore dari backup dan edit ulang lebih hati-hati

3. "Perubahan tidak terlihat di browser"
   ‚ûú Tekan Ctrl + F5 (hard refresh)
   ‚ûú Clear cache browser

4. "Karakter aneh muncul (ÔøΩ, ‚ñ°, dll)"
   ‚ûú Encoding file berubah
   ‚ûú Pastikan save dengan encoding UTF-8

5. "Website tidak bisa dibuka"
   ‚ûú Pastikan file bernama "index.html" bukan "index.html.txt"
   ‚ûú Buka dengan double-click atau drag ke browser
